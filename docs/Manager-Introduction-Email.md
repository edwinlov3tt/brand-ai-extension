## What It Does

Brand AI Extension drastically reduces the time it takes to create brand profiles and generate ad copy from **2-3 hours down to 15-30 minutes**. Here's what's currently working:

### Core Features

**1. Instant Brand Extraction**
- Automatically detects and extracts brand colors, fonts, and logos from any website
- Smart weighting system prioritizes important elements (headers, CTAs) over backgrounds
- Captures brand voice, positioning, and target audience details

**2. AI-Powered Ad Copy Generation**
- 30+ ad copy templates across all major platforms:
  - Google Search, Facebook, Instagram, LinkedIn, TikTok, Snapchat, Pinterest, Twitter
  - Email marketing, native ads, and more
- Generate 1-10 variations per campaign objective
- Multi-component ads (headline + body + CTA) in one click
- Optional emoji support with smart placement

**3. Product/Service Context**
- Save specific products or services from prospect websites
- Generate targeted ad copy for specific offerings (not just generic brand copy)
- AI automatically incorporates value props, features, and benefits

**4. Management Dashboard**
- Organized view of all saved brands
- Complete ad copy history with filtering
- One-click copy for single variations or entire multi-component sets
- Visual brand profile viewer

## How to Install (For Now)

Since we're not on the Chrome Store yet, here's how to get it running:

1. **Enable Developer Mode in Chrome:**
   - Open Chrome and go to `chrome://extensions/`
   - Toggle "Developer mode" ON (top right corner)

2. **Install the Extension:**
   - Click "Load unpacked"
   - Select the extracted extension folder from the zip file I'm sending
   - The extension icon should appear in your toolbar

3. **Start Using It:**
   - Click the extension icon on any website to open the side panel
   - Create your first brand profile and start generating ad copy
